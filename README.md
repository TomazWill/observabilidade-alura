## Passo a passo ##########

### Rodando docker
```sh

```

### Rodando aplicação (redis...)
```sh
cd prometheus-grafana && \
docker-compose up
```

### Rodando aplicação java
```sh
cd prometheus-grafana/app && \
sh start.sh
```
